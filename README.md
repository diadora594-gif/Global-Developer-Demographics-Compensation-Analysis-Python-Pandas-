# Global Developer Demographics & Compensation Analysis

An exploratory data analysis (EDA) of the Stack Overflow Global Developer Survey dataset using Python and Pandas. This project uncovers global tech ecosystem trends, housing formats, remote work distributions, and key drivers behind developer compensation.

## 🔧 Tech Stack
* **Language:** Python
* **Library:** Pandas (Data manipulation, descriptive statistics, and filtering)
* **Environment:** Jupyter Notebook / Google Colab

## 📊 Key Analytical Tasks Addressed
1. **Dataset Volume Assessment:** Calculated total respondents to gauge survey scale.
2. **Data Completeness & Quality Audit:** Analyzed missing values using set intersections to isolate comprehensive responses.
3. **Statistical Deep-Dive:** Computed measures of central tendency (Mean, Median, Mode) for developer work experience.
4. **Workplace Trends:** Isolated and quantified the remote workforce segment.
5. **Language Popularity Tracking:** Processed multi-valued strings to measure Python's market penetration.
6. **Education Path Analysis:** Evaluated the footprint of online courses in developer training.
7. **Geographical Compensation Mapping:** Aggregated regional salary metrics (Mean/Median) specifically for Python engineers.
8. **Top-Earning Profiles:** Isolated high-income brackets and mapped their educational backgrounds.
9. **Age-Based Adoption Rates:** Tracked Python popularity across diverse age segments.
10. **High-Value Segment Extraction:** Filtered remote professionals within the top 75th percentile of global compensation to extract dominant industry sectors.
